# 复合题材-融合逻辑

## 题材组成
- 主题材：都市修仙（占比70%）
- 副题材：系统流 + 直播网红（占比30%）
- 占比建议：7:3（主线保持修仙逻辑，副线提供创新点）

## 融合目标
- 共同核心冲突：传统 vs 创新，隐秘 vs 公开
- 共同爽点目标：主角用现代方式破解修仙难题，打脸保守派
- 读者承诺：看到不一样的修仙，接地气，有共鸣

## 融合机制
- 规则兼容点：
  1. 灵气可作为直播"特效"，系统可作为修炼"外挂"
  2. 人气值可作为修炼资源，打赏可兑换现实好处
  3. 现代知识可辅助修仙，程序员思维解传统难题
- 冲突点与解决：
  1. **价值观冲突**：传统派认为修仙应隐秘庄重
     - 解决：通过公开比赛证明直播修仙的可行性
  2. **修炼方式冲突**：苦修 vs 直播互动
     - 解决：展示直播修炼的高效率，但承认根基问题
  3. **资源争夺冲突**：传统派垄断资源
     - 解决：主角开辟新资源渠道（人气兑换）
- 副题材介入条件（何时触发）：
  1. **系统任务**：每章都可能触发
  2. **直播需求**：每日必须直播2小时
  3. **人气里程碑**：每达到一定人气解锁新功能
  4. **特殊事件**：平台活动、粉丝互动、商业合作
- 不可混用的规则（必须隔离）：
  1. 传统修仙的"心境感悟"不能靠刷人气跳过
  2. 系统任务不能要求违背天道誓言的内容
  3. 直播内容需遵守现实法律和平台规则

## 主线/副线分工
- 主题材承担：
  1. 境界提升，实力增长
  2. 修仙界恩怨，势力斗争
  3. 大道感悟，心境成长
  4. 秘境探索，资源争夺
- 副题材承担：
  1. 日常直播，粉丝互动
  2. 系统任务，功能解锁
  3. 网红运营，商业变现
  4. 现代知识解修仙题
- 节奏安排（每 N 章一次副题材钩子）：
  1. 每1-2章：日常直播内容
  2. 每5章：系统重要任务
  3. 每10章：人气里程碑，解锁新功能
  4. 每20章：大型直播活动或商业合作

## 风险清单
- 风格割裂点：
  1. 修仙术语与网络用语混用可能违和
     - 规避：主角初期不熟练，后期找到平衡点
  2. 严肃战斗与直播弹幕同时出现
     - 规避：战斗时关闭弹幕显示，或作为特殊效果
- 设定冲突点：
  1. 传统修仙需要安静环境 vs 直播需要热闹互动
     - 规避：设计"专注模式"，直播时可屏蔽干扰
  2. 修仙寿命漫长 vs 网红生命周期短
     - 规避：主角定位为"长期修仙网红"，内容可持续
- 读者预期偏差点：
  1. 期待传统修仙的读者可能不适应
     - 规避：明确题材标签，前期做好铺垫
  2. 期待纯爽文的读者可能嫌修炼过程慢
     - 规避：保持爽点密度，每章都有小成就
- 规避办法：
  1. 前期做好题材预告
  2. 保持核心修仙逻辑不崩
  3. 平衡传统与创新元素

## 与创意约束对齐
- 反套路规则：修仙必须公开，打破"闷声发大财"
- 硬约束：系统强制任务，必须直播修炼
- 主角缺陷如何被放大：社交恐惧症被迫面对观众，理性思维被非理性任务挑战
