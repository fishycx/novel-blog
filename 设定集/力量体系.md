# 力量体系设定

## 体系公理
- 核心信条/定律：灵气即流量，修为即人气
- 代价规则：公开修炼，接受监督；人气越高，责任越大
- 公平性原则：系统绑定者可通过直播快速提升，但必须公开透明；传统修仙者靠天赋苦修，但可私下进行

## 体系类型
- 体系类型：双轨制（传统修仙 + 直播修仙）
- 典型境界链：炼气 → 筑基 → 金丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫
- 小境界划分：初期 → 中期 → 后期 → 巅峰

### 常见境界链参考（可选）
- 修真链：练气 → 筑基 → 金丹 → 元婴 → （后续自定义）

## 能力来源
- 能量/资源来源：
  1. 天地灵气（传统）
  2. 人气值（直播修仙特有）
  3. 信仰之力（粉丝崇拜）
- 训练/修炼方法：
  1. 吐纳打坐（传统）
  2. 直播互动（系统特有）
  3. 任务完成（系统奖励）
- 社会控制机制：
  1. 修仙者协会（官方组织）
  2. 直播平台监管
  3. 系统强制规则

## 等级体系
- 等级顺序：
  1. 炼气期（1-9层）：初步感知灵气，强化身体
  2. 筑基期（1-9层）：灵气液化，寿命200年
  3. 金丹期（1-9层）：凝聚金丹，可御剑飞行
  4. 元婴期（1-9层）：元婴出窍，神通初显
  5. 化神期（1-9层）：神识外放，感悟法则
  （后续境界随剧情解锁）

- 每级核心能力：
  - 炼气期：灵气感知，身体强化，基础法术
  - 筑基期：御物飞行，法术精通，寿命延长
  - 金丹期：金丹护体，神通初显，可开宗立派
  - 元婴期：元婴不灭，肉身可重塑，空间穿梭

## 晋级条件
- 资源要求：灵气浓度、丹药辅助、心境感悟
- 突破方式：
  1. 传统：闭关顿悟，丹药冲击
  2. 直播：人气达标，任务完成，观众助力
- 失败代价：修为倒退，心境受损，直播掉粉
- 越级代价：根基不稳，心魔易生，系统惩罚

## 资源系统
- 核心资源类型：
  1. 灵石（传统货币）
  2. 人气值（系统货币）
  3. 打赏礼物（现实货币）
  4. 丹药、功法、法宝
- 资源获取方式：
  1. 修炼吸收
  2. 直播赚取
  3. 任务奖励
  4. 交易购买
- 稀缺性规则：高阶资源有价无市，需要特殊渠道
- 替代路径：系统商城可用人气兑换，但价格昂贵

## 战斗规则
- 伤害与防御逻辑：灵气对撞，法宝护体，神识压制
- 战斗节奏特点：前期法术对轰，中期法宝比拼，后期法则对抗
- 克制/反制关系：
  1. 火克金，金克木，木克土，土克水，水克火
  2. 直播修仙怕信号干扰，传统修仙怕舆论压力
- 逃生/撤退机制：遁术、传送符、系统紧急保护

## 禁忌与限制
- 禁术/禁地：夺舍、血祭、搜魂；上古遗迹、禁地
- 高阶限制：高境界不得随意对低境界出手（天道约束）
- 硬限制：直播修仙必须公开，传统修仙可私下

## 体系漏洞与利用
- 漏洞：人气值可人为刷量，系统有BUG可卡
- 主角如何利用：用程序员思维找系统BUG，合理卡规则
- 反派如何反制：雇佣水军刷差评，黑客攻击直播间

## 反馈节奏
- 境界变化节点：
  - 每30章：小境界提升
  - 每100章：大境界突破
  - 关键节点：直播突破，观众见证
- 读者可感知的战力差距展示方式：
  1. 炼气打不过筑基（绝对压制）
  2. 金丹可秒杀筑基（境界鸿沟）
  3. 直播人气影响实战（特殊加成）

## 与创意约束对齐
- 反套路规则如何体现：修仙必须公开，打破传统神秘感
- 硬约束如何绑定体系：系统强制直播，传统派鄙视直播派，形成冲突
