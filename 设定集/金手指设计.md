# 金手指设计

## 设定定位
- 题材适配点：完美结合"修仙"与"网红"两大元素，符合现代读者喜好
- 主要作用：推进主线（强制任务驱动）、解决硬约束（经济压力）、放大爽点（直播打脸）
- 读者可见度：明牌（系统面板直接展示给读者）

## 常见类型参考（可选）
- 系统 / 空间 / 位面穿梭 / 重生记忆 / 传承 / 血脉 / 召唤

## 类型
系统面板（强制任务型 + 直播互动型）

## 核心功能
- 功能描述：
  1. **任务系统**：发布修仙相关的直播/短视频任务
  2. **人气兑换**：直播人气可兑换修为、功法、丹药
  3. **弹幕互动**：观众弹幕可触发特殊事件（福袋、挑战等）
  4. **数据分析**：分析直播数据，优化修仙效果
  5. **跨界商城**：用打赏收入购买修仙资源

- 可视化表现：
  - 半透明悬浮面板，只有主角可见
  - 类似直播后台的UI设计（观看人数、礼物榜、任务进度条）
  - 弹幕以灵气流形式飘过

- 触发条件：
  - 主角在失业绝望时，对手机发誓"只要能赚钱，什么都干"
  - 手机被雷劈中（实际是灵气潮汐），系统激活

## 获得方式
- 触发事件：2026年3月，全球灵气复苏前夕，主角手机被异常雷击
- 代价/限制：
  1. 必须完成每日/每周任务，否则扣除修为
  2. 修仙过程必须直播，不能私下修炼
  3. 系统会发布"社死"任务考验心态
- 不可逆代价：绑定灵魂，无法卸载；修炼进度公开透明

## 使用规则
- 冷却/次数限制：
  - 每日直播至少2小时
  - 每周必须完成1个主线任务
  - 特殊任务有倒计时

- 禁止事项：
  1. 向非绑定者透露系统存在
  2. 故意摆烂、消极直播
  3. 利用修仙能力违法犯罪

- 失败惩罚：
  - 轻度：扣除人气值
  - 中度：修为倒退
  - 重度：触发"社死"惩罚任务

- 反制方式（敌人如何克制）：
  1. 黑客攻击直播间
  2. 买水军刷差评降低人气
  3. 物理干扰直播设备
  4. 举报直播间违规

## 升级路线
- 初始形态：Lv.1 新手主播（基础功能）
  - 任务发布、人气兑换、基础数据分析

- 中期提升：Lv.10 网红修士（解锁新功能）
  - 弹幕互动事件、跨界商城、粉丝团系统
  - 可接广告任务，用商家产品修仙

- 最终形态：Lv.50 跨界道祖（完全体）
  - 开辟专属修仙直播间（小世界）
  - 可收徒建立"直播修仙宗门"
  - 系统与自身大道融合

## 爽点嵌入
- 获得爽：失业青年突然获得改变命运的机会
- 成长爽：看着人气值上涨，修为提升，打脸看不起的人
- 使用爽：直播中当场突破，观众打赏如雨；用现代知识破解修仙难题

## 反馈节奏
- 关键反馈节点（每N章必须体现）：
  - 每3章：小任务完成，修为小提升
  - 每10章：人气里程碑，解锁新功能
  - 每30章：系统升级，能力质变

- 代价兑现节点：
  - 第5章：第一次任务失败，被惩罚当街跳舞
  - 第25章：重要直播被干扰，修为倒退
  - 第80章：系统强制发布危险任务

- 反转节点：
  - 第15章：发现人气可兑换现实货币
  - 第50章：弹幕中出现真正修仙者
  - 第120章：系统透露灵气复苏真相

## 与创意约束对齐
- 反套路规则如何体现：修仙必须公开直播，打破"闷声发大财"传统
- 硬约束如何绑定金手指：经济压力→需要直播赚钱；社交恐惧→必须面对观众
- 主角缺陷如何被放大/修复：理性思维被系统强制任务挑战，逐渐学会变通
